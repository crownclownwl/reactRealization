# 这是一个自学项目

## 常用的 type 值包括如下:

- feat: 添加新功能
- fix: 修复 Bug
- chore: 一些不影响功能的更改
- docs: 专指文档的修改
- perf: 性能方面的优化
- refactor: 代码重构
- test: 添加一些测试代码等等

## 目录结构说明：

```shell
.
├── node_modules              # 项目依赖包
├── package.json              # 项目依赖配置
├── packages                  # 代码
│ └── test.js
├── pnpm-lock.yaml
├── pnpm-workspace.yaml
├── prettier.json             # prettier 配置文件
├── ReadMe.md
├── scripts                   # 项目用到的所有工具的配置脚本
│ └── rollup                  # 所有rollup的配置
└── tsconfig.json             # ts配置文件
```
